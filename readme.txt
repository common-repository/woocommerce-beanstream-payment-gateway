== Description ==
The WooCommerce Beanstream Payment gateway plugin allows you to accept online payments in your WordPress WooCommerce store. Download the WooCommerce Beanstream plugin and start accepting payments from all major brands with your Beanstream account: Visa, MasterCard, American Express, Discover and more.

<strong>WooCommerce plugin</strong><br>
WooCommerce plugin is an e-commerce toolkit that helps you sell anything. Transform your WordPress website into online eCommerce store. Delivering enterprise-level quality, features and add ons to help you control your users experience.

This plugin works with the WooCommerce plugin and allows you to accept payments via your online store.

Beanstream payment gateway is for canada so it will be use in on canadian doller.

Sure your woocommerce currency in canadian doller.



== Installation ==
<strong>Simply follow these steps to install:</strong><br>
1. Unzip the file.<br>
2. Upload the "woocommerce-beanstream-gateway" folder to your WordPress Plugins directory.<br>
3. Login to your WordPress Admin, then go to Plugins and activate the "WooCommerce - Beanstream Payment" plugin<br>
4. Within the WordPress Admin, go to WooCommerce >> Settings, then click on the Payment Gateways tab, then click on the Beanstream link.<br>
5. Enter the proper information...<br>

<strong>To use in "Live Mode"</strong>
	<br>
1. Gather your QuickLink API Login and Password as explained on the following article: http://support.beanstream.com/<br>
2. Login to your WordPress Admin.<br>
3. Go to WooCommerce >> Settings >> Beanstream.<br>
4. Un-check "Enable Beanstream Test".<br>
5. Enter your Marchent ID (Provided by beanstram).<br>
6. Enter your Passcode (Provided by beanstram).<br>
7. Click "Save". Your website/shopping cart is now setup to process transactions via your Beanstream acount. We recommend that you process a test transaction, then login to your Beanstream account to confirm that it appears in your Current Unsettled Batch ( Transaction History).<br>
   
<strong> To use in "Test Mode" </strong><br>
1. Login to your WordPress Admin.<br>
2. Go to WooCommerce >> Settings >> Beanstream.<br>
3. Check the box for "Enable Beanstream Test".<br>
4. Use the following API Username = pj-ql-01<br>
5. Use the following API Password = pj-ql-01p<br>
